# Postman Collections

This directory contains API test collections for the Job Seeker module.

## Setup Instructions

1. Import the collection file into Postman
2. Set up your environment variables in Postman
3. Configure authentication tokens as needed

## Environment Variables Required

- `baseUrl`: API base URL (e.g., http://localhost:4000)
- `authToken`: JWT token for authenticated requests
- `userId`: Test user ID for profile operations

## Note

Actual collection and environment files are excluded from version control for security reasons.
